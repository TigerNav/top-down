1. change into this directory. For example:
   cd "path to this directory"

2. install cmake
   for linux users :
      Debian based : sudo apt-get install cmake
      arch based   : sudo pacman -S cmake
      fedora and red hat based : sudo dnf install cmake

3. use command :
   cmake ../CMakeLists.txt

4. then use make file by running :
   make

5. execute the game by running :
   ./main
